/**
 * @Comment
 *
 * Calculate:
 *  Number of boxes required to hold all the items.
 *  Number of items in the last box.
 *	Serial numbers for the items.
 *	The number of the box. e.g. Box x of y.
 *
 */


/**
 * @Comment
 *
 * The number of boxes required equals the total number of items divided by the number
 * of items that a box will hold. If there is a remainder add another box. The
 * remainder will be the number of items in the last box.
 *
 */

 *
